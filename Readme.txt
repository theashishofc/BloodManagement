1. Run Xampp. 

2. Extract BloodManagement zip file

3. Copy BloodManagement folder into your xampp >> htdocs folder.

4. Open localhost/phpmyadmin

5. Create new database named bms_db.

6. Go to import and select file from htdocs/BloodManagement/database/bms_db.sql  and click on Go below.

7. Now run the webapp  localhost/BloodManagement
